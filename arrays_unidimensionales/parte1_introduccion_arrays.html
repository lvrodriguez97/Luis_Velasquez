Primeros Pasos con Arrays (Arreglos)

Para iniciar con el tema de Arrays en Java, lo más fácil es imaginar un array como una caja de casilleros numerados donde cada casillero guarda un único dato, y todos los casilleros deben pertenecer estrictamente al mismo tipo de dato.

1 Declarar e Inicializar (Saber el tamaño de antemano)

Si sabes cuántos casilleros necesitas pero aún no tienes los datos listos, los creas definiendo su capacidad. Java los rellenará automáticamente con ceros (`0`) si son de tipo numérico.

public class EstructuraBasica { public static void main(String[] args) { // Creamos un array de enteros con espacio para 4 elementos int[] edades = new int[4]; // Guardamos datos en cada casillero usando su ÍNDICE (coordenada) // ¡Recuerda que el primer casillero siempre es el 0! edades[0] = 15; // Primer casillero edades[1] = 20; // Segundo casillero edades[2] = 18; // Tercer casillero edades[3] = 25; // Cuarto casillero (Último) // Acceder y mostrar un dato específico System.out.println("La edad en el segundo casillero es: " + edades[1]); } }

2 Inicialización Directa (Con datos listos)

Si ya conoces de antemano los valores que van a ir dentro del array, puedes declararlo y rellenarlo todo en una sola línea usando llaves {} sin usar la palabra clave new.

public class ArrayDirecto { public static void main(String[] args) { // Declaración rápida con datos predefinidos String[] nombres = {"Ana", "Juan", "Pedro"}; // Saber el tamaño total del array con la propiedad .length System.out.println("Cantidad de personas: " + nombres.length); // Mostrar el primer elemento System.out.println("Primer nombre: " + nombres[0]); } }

3 Recorrer un Array con un Bucle for Tradicional

Para no escribir System.out.println por cada casillero de forma manual, usamos un ciclo for que use una variable contadora (i) para moverse automáticamente por los índices desde el 0 hasta el final.

public class RecorrerArray { public static void main(String[] args) { int[] notas = {90, 85, 78, 95, 100}; System.out.println("--- Lista de Notas ---"); // i empieza en 0 y va aumentando mientras sea menor al tamaño del array for (int i = 0; i < notas.length; i++) { System.out.println("Índice " + i + " -> Nota: " + notas[i]); } } }

4 El "Error Clásico" que debes evitar

Cuando se empieza a programar con arrays, el error más común es intentar acceder o escribir en un casillero que no existe en la memoria física asignada.

public class ErrorComun { public static void main(String[] args) { int[] numeros = {10, 20, 30}; // Índices válidos: 0, 1 y 2 // ERROR: El tamaño es 3, pero el índice máximo es 2. // La siguiente línea romperá el programa en tiempo de ejecución: System.out.println(numeros[3]); } }

⚠️ Tres Reglas de Oro para Memorizar
Base Cero

El conteo de posiciones de los casilleros en memoria empieza siempre en 0, no en 1.

Tamaño Fijo

Una vez que defines que un array mide 4, no puedes agrandarlo a 5 en mitad de la ejecución.

Último Índice

La última posición disponible de un array siempre se encuentra en (longitud - 1).

5 Buscar el Valor Máximo y Mínimo

Recorrer el array una sola vez comparando cada elemento contra una variable de registro es el patrón fundamental. La clave: inicializar con numeros[0], nunca con 0. Si todos los valores del array fueran negativos, partir de 0 daría un resultado incorrecto.

public class BuscarMaxMin { public static void main(String[] args) { int[] numeros = {34, 7, 23, 32, 5, 62}; int max = numeros[0]; // punto de partida: primer elemento int min = numeros[0]; for (int i = 1; i < numeros.length; i++) { if (numeros[i] > max) max = numeros[i]; if (numeros[i] < min) min = numeros[i]; } System.out.println("Máximo: " + max); // 62 System.out.println("Mínimo: " + min); // 5 } }
Traza paso a paso — {34, 7, 23, 32, 5, 62}
6 Invertir un Array (Técnica In-Place)

Dos punteros avanzan desde los extremos hacia el centro intercambiando elementos. Solo se necesita una variable temp para no perder datos al sobrescribir. El bucle trabaja exactamente length / 2 veces, sin tocar el elemento central en arrays de longitud impar.

public class InvertirArray { public static void main(String[] args) { int[] arr = {1, 2, 3, 4, 5}; int izq = 0; int der = arr.length - 1; while (izq < der) { int temp = arr[izq]; // guardar antes de sobrescribir arr[izq] = arr[der]; arr[der] = temp; izq++; der--; } // resultado: {5, 4, 3, 2, 1} } }
[TRAMPA CLÁSICA] Si el ciclo recorriera el array completo (sin que izq y der se crucen), los elementos se invertirían dos veces y el array regresaría a su estado original.
Traza paso a paso — {1, 2, 3, 4, 5}
7 Detectar Duplicados (Doble Bucle)

Cada elemento se compara contra todos los que vienen después de él. El bucle interno arranca en i + 1 por dos razones: evitar comparar un elemento consigo mismo y evitar reportar el mismo duplicado dos veces.

public class DetectarDuplicados { public static void main(String[] args) { int[] datos = {2, 3, 2, 5, 3, 8}; for (int i = 0; i < datos.length; i++) { for (int j = i + 1; j < datos.length; j++) { if (datos[i] == datos[j]) { System.out.println("Duplicado: " + datos[i]); } } } } }
Traza paso a paso — {2, 3, 2, 5, 3, 8}
