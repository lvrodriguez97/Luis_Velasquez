Ejercicios Resueltos con Funciones

En Java, los procesos lógicos no se escriben sueltos. Se encapsulan en funciones que reciben el array, lo recorren mediante bucles y devuelven un único resultado estructurado.

Ejercicio 1: Encontrar el número más alto
Método que recorre el array comparando elementos para extraer el valor máximo.
public static int obtenerMaximo(int[] arr) { int max = arr[0]; for (int i = 1; i < arr.length; i++) { if (arr[i] > max) { max = arr[i]; } } return max; }
Array de entrada: 'arr'

• i = -
• max = -

Ejercicio 2: Calcular el promedio de notas
Método de acumulación (sumatoria) y división final por la propiedad matemática .length.
public static double calcularPromedio(int[] notas) { int suma = 0; for (int n : notas) { suma += n; } return (double) suma / notas.length; }
Array de entrada: 'notas'
Variables de acumulación:
• n (Nota actual) = -

Ejercicio 3: Buscar un elemento (¿Existe el ID en el sistema?)
Algoritmo de búsqueda que retorna el índice donde se encuentra el objetivo, o -1 si no existe. Objetivo buscado: 99.
public static int buscarElemento(int[] arr, int objetivo) { for (int i = 0; i < arr.length; i++) { if (arr[i] == objetivo) { return i; // Retorna la coordenada } } return -1; }
