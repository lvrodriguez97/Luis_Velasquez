Arrays Bidimensionales (Matrices)

Para comprender los Arrays Bidimensionales en Java, la abstracción más efectiva consiste en estructurarlos como una tabla de datos organizada por filas y columnas. Conceptualmente, en Java se definen y manejan como un arreglo que contiene otros arreglos dentro de sí.

1 Declaración y Dimensionamiento Espacial

Cuando se define una matriz indicando sus dimensiones explicitas, el primer par de corchetes determina el total de filas, mientras que el segundo par determina las columnas asignadas a cada una.

public class MatrizBasica { public static void main(String[] args) { // Declaración de una matriz con 3 filas y 4 columnas int[][] matriz = new int[3][4]; // Asignación de datos utilizando las coordenadas explícitas [fila][columna] matriz[0][0] = 5; // Primera fila, primera columna matriz[0][3] = 21; // Primera fila, última columna matriz[2][2] = 45; // Última fila, tercera columna } }

2 Interpretación de la Propiedad .length

Al tratar con matrices, la propiedad .length cambia su comportamiento dependiendo del nivel de anidamiento al que se aplique:

matriz.length

Retorna la cantidad total de filas que componen la estructura bidimensional.

matriz[f].length

Retorna la cantidad de columnas contenidas específicamente en la fila indicada por el índice f.

[ESTRUCTURAS IRREGULARES] En Java, cada fila puede poseer un tamaño de columnas diferente debido a que una matriz es un arreglo de arreglos independientes. A este tipo de matriz se le denomina Jagged Array.
3 Recorrido Completo Mediante Ciclos Anidados

Para procesar todos los elementos, se ejecuta un bucle externo encargado de controlar el desplazamiento entre filas y un bucle interno asignado al recorrido de las columnas de dicha fila.

public class IterarMatriz { public static void main(String[] args) { int[][] datos = { {1, 2, 3}, {4, 5, 6} }; for (int f = 0; f < datos.length; f++) { for (int c = 0; c < datos[f].length; c++) { System.out.print(datos[f][c] + " "); } System.out.println(); // Salto al finalizar cada fila } } }

4 Desbordamiento en Matrices

El error de índice fuera de rango ocurre de forma independiente tanto si la coordenada de la fila como la coordenada de la columna superan los límites establecidos por sus respectivas dimensiones.

int[][] muestra = new int[2][2]; // Índices válidos: 0 y 1 // Error fatal de ejecución: // La fila 0 existe, pero la columna 2 está fuera de rango. System.out.println(muestra[0][2]);
[EXCEPCIÓN EN MATRICES] Provocar este escenario lanzará un ArrayIndexOutOfBoundsException, interrumpiendo el flujo del programa inmediatamente.
